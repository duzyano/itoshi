# Happy Herbivore – Self Service Kiosk

## Samenvatting

Voor dit project heb ik een webapplicatie ontwikkeld voor een **self-service kiosk systeem** van Happy Herbivore. Met dit systeem kunnen klanten zelfstandig hun bestelling plaatsen via een digitaal scherm.

Tijdens het project heb ik eerst onderzoek gedaan naar hoe een kiosk-bestelsysteem werkt en wat gebruikers nodig hebben. Op basis daarvan heb ik een planning gemaakt, een functional design opgesteld en een database ontworpen om producten en bestellingen op te slaan.

Daarna heb ik de **front-end** van de applicatie ontwikkeld, waarbij de gebruiker door het volledige bestelproces kan navigeren. Voor de interface heb ik moderne webtechnologieën gebruikt zoals **React** en **Tailwind CSS**.

Vervolgens heb ik een **API en backend** ontwikkeld en deze gekoppeld aan de front-end, zodat de applicatie data kan ophalen en bestellingen kan verwerken via de database.

Tot slot heb ik de applicatie getest op functionaliteit, compatibiliteit en performance. Op basis van deze tests zijn mogelijke verbeteringen en optimalisaties voorgesteld.

Het resultaat is een werkend prototype van een **self-service kiosk webapplicatie** waarmee het bestelproces voor klanten eenvoudiger en efficiënter wordt.
